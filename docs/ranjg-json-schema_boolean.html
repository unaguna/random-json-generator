Boolean GenerationÂ¶

When boolean is adopted as type, ranjg.gen returns True or False randomly. Then the other keywords are ignored.

>>> import ranjg
>>> schema = { 'type': 'boolean' }
>>> generated = ranjg.gen(schema)  # -> returns True or False
>>> type(generated)
<class 'bool'>
